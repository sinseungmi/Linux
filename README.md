> 
> ##### YUM(Yellow Dog update Modified) - 프로그램(패키지) 설치 관리 도구
> ```
> yum [옵션] [명령] [패키지명]    
> ```
> 1. YUM 패키지 설치/삭제 명령
> ```
> yum install 패키지명 // 패키지 설치
> yum remove 패키지명 // 패키지 삭제
> ```
> 2. 패키지 모듈을 update하는 명령. (단, 설치되어 있지 않으면 but not installed 메시지가 뜹니다.)
> ```
> sudo yum update lrzsz
> ```
